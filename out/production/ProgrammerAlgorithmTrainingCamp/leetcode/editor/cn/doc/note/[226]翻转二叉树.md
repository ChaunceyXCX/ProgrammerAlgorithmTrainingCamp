## 226.翻转二叉树

### 递归法

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        invertChild(root);
        return root;
    }

    public void invertChild(TreeNode cur){
        // 递归终止条件
        if (cur == null) return;
        TreeNode temp = cur.left;
        cur.left = cur.right;
        cur.right = temp;
        invertChild(cur.left);
        invertChild(cur.right);
    }
}
```
